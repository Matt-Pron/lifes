# Lifes

**Lifes** is a TUI TTRPG Health Points tracker for Linux.

It's made in c++, using ncurses.

## Building

### Build

```
make
```

### Run

```
./bin/lifes
```

### Clean Build

```
make clean
```

## Example tables

To fill the tables with simple example data use:

```
cp ./assets/data ./bin/
```
